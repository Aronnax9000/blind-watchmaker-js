Monochrome Geneboxes

In JavaScript, each genebox is represented as a separate object, collected into an ordered collection called "geneboxes" in the code. A genebox collection keeps track of a single biomorph genome. Each child genebox shares access to the genome for purposes of displaying and modifying the genome's values. Each genebox keeps track of its order within the genebox collection. The collection is 0-indexed, unlike Pascal arrays, which are 1-indexed, so genebox 0 in the Monochrome Biomorph is assigned to Gene 1, etc.

  • geneboxIndex: which genebox this is, in display order from 0-whatever.
    • 0: Gene 1, Gradient Gene 1
    • 1: Gene 2, Gradient Gene 2
    • 2: Gene 3, Gradient Gene 3
    • 3: Gene 4, Gradient Gene 4
    • 4: Gene 5, Gradient Gene 5
    • 5: Gene 6, Gradient Gene 6
    • 6: Gene 7, Gradient Gene 7
    • 7: Gene 8, Gradient Gene 8
    • 8: Gene 9, Gradient Gene 9
    • 9: SegNoGene
    • 10: SegDistGene, Gradient Gene 10
    • 11: CompletenessType
    • 12: SpokesType
    • 13: TrickleGene
    • 14: MutProbGene
    • 15: MutSizeGene

Here are a couple of mocked up geneboxes. The numerical values are hard-coded HTML, as are the position of the gradient gene dots. There is one slight bit of interactivity: on a device with a visible pointer, the pointer should change to a left arrow if one hovers over the left third of a genebox, and a right arrow if over the right third. In the middle, a further division occurs into top, middle, and bottom for up arrow, equals, and down arrow cursors respectively. This is done using a stylesheet, with no code required.
